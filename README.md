# ryoma-code-assistant

code assistant：skills, mcp and so on

## Skills

- `git-commit` — conventional commit helper with diff analysis and staging guidance.
- `golang-review` — Go-specific review skill that invokes `golang-reviewer` for Go diffs and PRs.
- `golang-patterns` — idiomatic Go patterns, error handling, concurrency, package design, and performance guidance.
- `golang-testing` — Go testing patterns for table-driven tests, subtests, benchmarks, fuzzing, coverage, and TDD.

## Agents

- `golang-reviewer` — senior Go code reviewer focused on idioms, concurrency, errors, performance, and security.
